# C51 单片机开发库 🚀

> 适用于 **STC89C52 / AT89C51** 等经典 51 内核单片机，提供轻量级硬件驱动与开发笔记。

---

## 📁 项目结构
```
├── src/           # 核心驱动代码（.c/.h）
├── docs/          # 开发笔记与文档
│   └── note.md    # 调试记录、使用说明
├── .gitignore     # 自动忽略 Keil 编译产物
└── README.md      # 项目说明
```

---

## ✨ 特性
- 项目文件夹分别叫buzzer/buzzer和AT24C02。
- 结尾两个视频是代码实现的演示。
- 纯 C51 编写，无冗余依赖，适配 51 单片机有限资源
- `.gitignore` 已配置，自动过滤 `*.obj`/`*.lst`/`*.hex` 等编译文件
- 配套 Markdown 笔记，记录开发思路与调试踩坑经验
- 代码风格简洁，便于二次开发与移植

---

## 🛠️ 快速开始
1.  **环境准备**：Keil uVision4/5（C51 版本）
2.  **使用流程**：
    - 将 `src/` 下的 `.h`/`.c` 文件添加到你的 Keil 工程
    - 在代码中包含头文件：`#include "xxx.h"`
    - 编译下载到 51 开发板运行

---

## 📝 文档说明
- `docs/note.md`：记录开发过程中的问题、调试方法与版本迭代
- 所有源码均附带注释，关键逻辑清晰易懂

---

## ⚠️ 注意事项
- 编译产生的中间文件（如 `.obj`/`.hex`/`.uvopt`）不会被上传到 GitHub
- 延时函数需根据实际晶振频率（如 11.0592MHz）调整参数
- IO 口映射请对照你的开发板原理图修改
