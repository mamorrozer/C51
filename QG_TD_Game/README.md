# QG 51 塔防游戏（简化版植物大战僵尸）

## 1. 硬件连接建议
- LCD12864（并口模式）：数据口 P0，RS=P2.3，RW=P2.4，EN=P2.2
- HW-504 矩阵键盘：P1 口扫描
- 蜂鸣器：P3.7
- AT24C02：SCL=P2.1，SDA=P2.0

## 2. 按键映射
- 2 / 8 / 4 / 6：光标上/下/左/右
- 5：放射手（消耗40资源）
- 9：放坚果墙（消耗30资源）
- 13：暂停/继续
- 16：返回（在游戏中可直接结束）

## 3. 游戏规则
- 3 条轨道，敌人从右向左移动
- 植物：
  - `W` 坚果墙：高血量，专门阻挡
  - `S` 射手：会自动发射 `*` 子弹
- 敌人：
  - `z` 普通敌人（慢，血高）
  - `Z` 快速敌人（快，血低）
- 子弹与敌人同格时命中，敌人血量归零后消失并加分
- 敌人到达最左侧会扣基地生命 `H`
- 生存到目标时间判定胜利，生命归零判定失败

## 4. 编译说明（Keil C51）
1. 新建 Keil 工程（STC89C52/AT89C51）
2. 把本目录全部 `.c/.h` 加入工程
3. `main.c` 作为入口编译下载

## 5. 局部刷新策略
- `ui.c` 内维护 `screen_now` 与 `screen_old`
- 每次只写入变化字符，减少闪屏
